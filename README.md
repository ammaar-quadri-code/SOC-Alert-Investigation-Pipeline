# SOC Investigation and Threat Hunting Lab

## Overview

This repository contains cybersecurity projects focused on Security Operations Center (SOC) investigations, alert analysis, threat hunting, and security automation.

The objective of this repository is to demonstrate how security alerts can be transformed into actionable investigations through structured analysis, indicator extraction, enrichment, detection logic, and incident triage workflows.

The project simulates real-world activities performed by SOC analysts when investigating suspicious events, validating alerts, correlating evidence, and generating findings for incident response teams.

---

## Objectives

Modern organizations generate thousands of security events daily from multiple security sources, including endpoint agents, intrusion detection systems, firewalls, SIEM platforms, and security monitoring tools.

The primary objectives of this project are to:

* Analyze security alerts
* Extract Indicators of Compromise (IOCs)
* Enrich findings with contextual information
* Correlate suspicious activities
* Prioritize incidents based on risk
* Generate investigation summaries
* Reduce analyst workload through automation
* Improve investigation efficiency and consistency

This repository serves as a practical learning platform for SOC operations, threat hunting methodologies, and security automation concepts.

## Features

### Alert Parsing

Processes incoming security alerts and converts them into structured information suitable for analysis.

### IOC Extraction

Identifies and extracts indicators such as:

* IP Addresses
* Domains
* URLs
* File Hashes
* User Accounts
* Hostnames

### Threat Intelligence Enrichment

Adds contextual information to extracted indicators to improve investigation quality and analyst decision-making.

### Detection Logic

Applies investigation rules and analytical workflows to determine whether activity may be suspicious or malicious.

### Sigma Rule Correlation

Maps security events against Sigma-based detection logic for additional validation and evidence gathering.

### Automated Triage

Assists in prioritizing alerts based on severity, confidence, risk, and available evidence.

### Investigation Reporting

Generates structured summaries to support SOC analysts during incident investigations.
